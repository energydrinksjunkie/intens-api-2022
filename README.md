# Intens Api DevOps
Uputstvo za pokretanje.

## Pokretanje servisa
`docker run --rm -p 8080:8080 energydrinksjunkie/intens-praksa`

[Adresa servisa](http://localhost:8080)

## Nakon push-a pokrenuti
`docker rmi energydrinksjunkie/intens-praksa`
`docker run --rm -p 8080:8080 energydrinksjunkie/intens-praksa`
